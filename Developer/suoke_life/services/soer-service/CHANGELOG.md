# 索儿智能体服务 (Soer Agent Service) 变更日志

## [1.1.0] - 2023-04-04

### 新增功能
- 集成知识库和知识图谱服务，实现知识增强功能
- 添加 `/api/v1/knowledge` 端点，支持知识库搜索、图谱查询和可视化
- 增强健康洞察和生活建议，通过知识库和知识图谱提供更丰富的信息
- 新增知识图谱可视化数据接口，支持健康知识图谱导航
- 实现中医知识集成，整合中医理论与现代健康理念

### 优化
- 重构代理服务架构，添加知识集成服务
- 改进智能体响应生成，增加知识支持
- 优化知识搜索性能，添加缓存机制
- 增强健康建议的可解释性，提供知识引用

### 修复
- 修复启动时服务集成初始化的错误
- 改进错误处理和日志记录

## [1.0.0] - 2023-03-26

### 初始版本
- 发布索儿智能体服务首个正式版本
- 实现健康数据分析功能
- 实现生活建议推荐功能
- 实现全态势健康感知功能
- 实现环境感知功能
- 实现智能体协作功能
- 提供基本的API端点和服务健康检查
- 实现与健康服务、生活记录服务、RAG服务、AI代理服务和感知服务的集成
