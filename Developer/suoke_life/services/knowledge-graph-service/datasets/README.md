# 索克生活知识图谱数据集

此目录包含用于构建和训练知识图谱的数据集。

## 目录结构

```
datasets/
├── tcm/              # 中医特色数据集
│   ├── herbs/        # 中药材数据
│   ├── formulas/     # 方剂数据
│   ├── syndromes/    # 证型数据
│   ├── acupoints/    # 穴位数据
│   ├── meridians/    # 经络数据
│   └── diagnostics/  # 诊断数据
├── multimodal/       # 多模态数据集
│   ├── images/       # 图像数据集
│   ├── audio/        # 音频数据集
│   └── text/         # 文本数据集
└── health/           # 健康数据集
    ├── genomic/      # 基因组数据
    ├── environmental/# 环境健康数据
    ├── mental/       # 心理健康数据
    └── medical/      # 医学健康数据
```

## 数据集版本控制

所有数据集文件采用以下命名规范：
`{dataset_name}_v{version}_{type}.{ext}`

例如：
- `tcm_herbs_v1.0_raw.json`
- `genomic_variants_v2.1_processed.csv`

## 使用指南

1. 数据集加载：使用 `DatasetLoader` 类加载数据集
2. 知识图谱构建：使用 `KnowledgeGraphBuilder` 从数据集构建图谱
3. 数据集更新：运行 `scripts/update_datasets.sh` 获取最新数据集

## 数据质量控制

所有数据集经过以下质量控制流程：
1. 数据清洗
2. 格式验证
3. 语义一致性检查
4. 标签规范化
5. 专家审核

## 训练集与测试集

对于机器学习模型训练，数据集按照 70:15:15 的比例分为训练集、验证集和测试集。

## 数据来源

- 中医数据来源：经典医籍、临床记录、专家知识
- 多模态数据：医学图像库、健康语音数据库
- 健康数据：匿名化医疗记录、公开健康数据集
