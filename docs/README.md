## 🛠️ 技术栈概览

### 后端技术
- **语言**: Python 3.13.3
- **框架**: FastAPI
- **包管理**: UV
- **数据库**: PostgreSQL, Redis, MongoDB, InfluxDB
- **搜索引擎**: Elasticsearch
- **数据处理**: 批处理和实时处理服务
- **AI/ML**: MLflow, PyTorch, TensorFlow

### 前端技术
- **框架**: React Native
- **语言**: TypeScript
- **状态管理**: Redux Toolkit
- **UI组件**: React Native Elements

### 基础设施
- **容器化**: Docker
- **编排**: Kubernetes
- **服务网格**: Istio
- **监控**: Prometheus + Grafana
- **日志**: ELK Stack
- **CI/CD**: GitHub Actions

### 质量保证
- **代码质量**: SonarQube
- **安全扫描**: Snyk
- **性能测试**: K6
- **测试框架**: pytest, Jest
