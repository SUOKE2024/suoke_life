## 1. 产品介绍

### 1.1 产品概述
索克生活是一个智能化生活助手平台，通过AI技术为用户提供个性化的生活方式指导和优化建议。

### 1.2 核心功能
- 智能助手系统
- 生活方式引擎
- 多维度进度追踪
- 隐私数据保护

### 1.3 特色亮点
- 场景化服务
- 情感化交互
- 个性化推荐
- 本地化处理

## 5. 常见问题

### 5.1 账号相关
```
Q: 如何修改密码？
A: 进入"设置-账号安全-密码修改"，按提示操作。

Q: 如何绑定/更换手机号？
A: 进入"设置-账号安全-手机绑定"，按提示操作。

Q: 忘记密码怎么办？
A: 点击登录页"忘记密码"，通过手机号/邮箱重置。
```

### 5.2 功能相关
```
Q: 如何选择合适的助手？
A: 根据场景需求选择：
   - 日常生活问题找小艾
   - 学习成长问题找老克
   - 工作事务找小克

Q: 数据如何同步？
A: 开启"设置-数据同步"，选择需要同步的数据类型。

Q: 如何保护隐私数据？
A: 系统默认本地加密存储，可在隐私设置中管理数据访问权限。
```
