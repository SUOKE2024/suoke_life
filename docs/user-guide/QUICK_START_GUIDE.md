3. 填写基本信息：

### 2. 认识四位智能体

#### 🌸 小艾 (XiaoAi) - 您的健康助手
- **专长**：日常健康咨询、症状初步评估、健康建议
- **适用场景**：
  - 感觉身体不适，想要初步了解可能原因
  - 需要日常健康建议和生活指导
  - 紧急情况下的初步处理建议

**使用示例**：
```
用户：我最近总是感觉疲劳，睡眠质量也不好，该怎么办？
小艾：根据您的症状，可能与以下因素有关：
1. 睡眠质量问题 - 建议调整作息时间
2. 工作压力过大 - 适当放松和减压
3. 营养不均衡 - 检查饮食结构
建议您先尝试改善睡眠环境，如有持续不适请咨询专业医生。
```

#### 🍃 小克 (XiaoKe) - 您的中医顾问
- **专长**：中医体质分析、中药调理、养生指导
- **适用场景**：
  - 想了解自己的中医体质类型
  - 寻求中医调理方案
  - 学习中医养生知识

**使用示例**：
```
用户：我经常手脚冰凉，怕冷，精神不振，这是什么体质？
小克：根据您的症状描述，初步判断您可能是阳虚体质：
- 主要特征：畏寒怕冷、手足不温、精神不振
- 调理建议：
  1. 饮食：多食温热食物，如生姜、桂圆、羊肉
  2. 运动：适度有氧运动，避免大汗淋漓
  3. 作息：早睡早起，避免熬夜
建议进一步进行详细体质分析以制定个性化调理方案。
```

#### 👨‍⚕️ 老克 (LaoKe) - 您的专业医师
- **专长**：深度健康分析、疾病风险评估、专业医学建议
- **适用场景**：
  - 需要专业的健康分析和建议
  - 慢性疾病管理和监控
  - 健康风险评估和预防

**使用示例**：
```
用户：我有高血压，最近血压控制不太好，该怎么办？
老克：高血压管理需要综合考虑多个因素：
1. 药物治疗：
   - 请按医嘱规律服药，不可自行停药
   - 定期监测血压变化
2. 生活方式干预：
   - 低盐饮食（每日盐摄入<6g）
   - 适量运动（每周150分钟中等强度运动）
   - 控制体重，戒烟限酒
3. 定期随访：建议每月复查血压，每3个月复查血脂、肾功能
请及时就医调整治疗方案。
```

#### 👶 索儿 (Soer) - 您的儿童健康专家
- **专长**：儿童健康评估、发育指导、儿科问题咨询
- **适用场景**：
  - 儿童健康问题咨询
  - 生长发育评估
  - 儿童疾病预防和护理

**使用示例**：
```
用户：我家宝宝3岁了，最近咳嗽流鼻涕，食欲不振，怎么办？
索儿：根据症状描述，宝宝可能是普通感冒：
1. 护理建议：
   - 多喝温水，保持室内湿度
   - 清淡饮食，少量多餐
   - 充分休息，避免过度活动
2. 观察要点：
   - 体温变化（超过38.5°C需就医）
   - 精神状态（嗜睡、烦躁需注意）
   - 呼吸情况（呼吸急促需立即就医）
如症状持续3天以上或加重，请及时就医。
```

## 💡 使用技巧

### 如何获得更好的建议

1. **详细描述症状**：
   - ✅ 好的描述："我头痛3天了，主要是太阳穴附近胀痛，早上起床时最严重，持续2-3小时后缓解"
   - ❌ 不好的描述："我头痛"

2. **提供相关背景**：
   - 年龄、性别、职业
   - 既往病史和用药情况
   - 最近的生活变化

3. **及时反馈**：
   - 告诉智能体建议是否有效
   - 描述症状的变化
   - 提出进一步的疑问

### 智能体协作功能

当您的问题比较复杂时，可以选择"多智能体协作"功能：

1. 点击"协作诊断"
2. 选择参与的智能体（建议选择2-3个）
3. 详细描述您的情况
4. 获得综合性的分析和建议

**协作示例**：
```
问题：45岁男性，慢性疲劳，睡眠质量差，血压偏高

小艾建议：改善睡眠环境，规律作息，适度运动
小克建议：可能是肾阳虚，建议温补肾阳，食疗调理
老克建议：需要排查甲状腺功能，监测血压，制定综合管理方案

综合建议：
1. 立即：改善睡眠环境，监测血压
2. 短期：完善相关检查（甲状腺功能、血脂等）
3. 长期：中西医结合调理，定期随访
```

## ⚠️ 重要提醒

### 使用须知

1. **不能替代医生**：
   - 智能体提供的建议仅供参考
   - 严重症状请及时就医
   - 不要自行停药或改变治疗方案

2. **隐私保护**：
   - 您的健康信息受到严格保护
   - 不会泄露给第三方
   - 可以随时删除个人数据

3. **紧急情况**：
   - 遇到紧急情况请立即拨打120
   - 胸痛、呼吸困难、意识障碍等需立即就医
   - 智能体会提供紧急处理建议但不能替代急救

### 何时需要立即就医

- 持续胸痛或胸闷
- 严重呼吸困难
- 意识障碍或昏迷
- 严重外伤或出血
- 高热不退（>39°C）
- 剧烈腹痛
- 突然的严重头痛
- 言语不清或肢体无力
