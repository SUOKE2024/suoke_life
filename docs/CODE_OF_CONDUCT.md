# 行为准则

## 承诺

为了营造开放和友好的环境，我们作为贡献者和维护者承诺：

- 友好和包容的参与环境
- 尊重不同观点和经验
- 理性和建设性的反馈
- 关注社区的共同利益

## 标准

积极行为包括：
- 使用友好和包容的语言
- 尊重不同的观点和经验
- 理性地接受建设性批评
- 关注社区最大利益

不当行为包括：
- 使用带有性暗示的语言或图像
- 挑衅、侮辱或贬低性评论
- 公开或私下的骚扰
- 未经明确许可发布他人信息
- 其他不符合职业操守的行为

## 责任

项目维护者有权并有责任：
- 删除、编辑或拒绝不符合本行为准则的内容
- 暂时或永久封禁行为不当的贡献者

## 适用范围

本行为准则适用于：
- 项目代码仓库
- Issue 和 PR 讨论
- 项目社区活动
- 项目相关的社交媒体
