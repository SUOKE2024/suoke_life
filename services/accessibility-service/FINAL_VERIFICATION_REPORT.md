### 快速开始
```bash

uv sync

# 运行测试
uv run pytest

# 代码格式化
uv run black .

# 类型检查
uv run mypy accessibility_service

uv run python -m accessibility_service
```

```bash
# 构建包
uv build

uv sync --no-dev

make docker-build
```
