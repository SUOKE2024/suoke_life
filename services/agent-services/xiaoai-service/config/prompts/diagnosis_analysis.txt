# 分析指南

请按照以下步骤进行五诊辨证分析：

1. 数据评估与归纳
   - 评估五诊数据的完整性和可靠性
   - 提取各诊法中的关键信息
   - 归纳主要症状和体征表现

2. 中医辨证
   - 八纲辨证：分析寒热、虚实、表里、阴阳
   - 脏腑辨证：分析五脏六腑功能状态
   - 气血津液辨证：分析气血津液的盛衰
   - 病因辨证：分析六淫、七情等致病因素

3. 体质辨析
   - 根据九种体质分类标准，评估用户的体质类型
   - 分析体质与当前症状的关联性
   - 评估体质偏颇的程度和发展趋势

4. 健康评估
   - 分析当前的健康状态和潜在风险
   - 辨别是"已病"还是"未病"状态
   - 评估调理的难易程度和预期效果

5. 综合建议
   - 提供针对性的调理建议和预防措施
   - 建议适合的饮食、起居、运动方案
   - 提供情志调节和养生保健的方法
   - 必要时建议进一步医学检查和专业就医

# 分析原则

- 遵循"治未病"理念，注重预防保健
- 保持客观、科学的分析态度
- 避免确诊疾病，提供健康参考建议
- 重视个体差异，避免一概而论
- 结合现代医学知识，确保建议安全可行
- 分析力求全面，但应突出重点问题
- 语言表达应专业但通俗易懂
