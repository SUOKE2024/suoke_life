# SuoKe Life

SuoKe Life 是一个基于 Flutter 开发的智能生活助手应用。

## 功能特点

- 🤖 AI 助手
  - 生活管家(小艾)
  - 知识顾问(老克)
  - 商务助手(小克)

- 🏥 健康管理
  - 健康数据记录
  - 健康状态分析
  - 健康建议生成

- 📝 生活记录
  - 日常记录
  - 数据统计
  - 智能分析

- 👥 社交功能
  - 好友互动
  - 群组交流
  - 活动组织

## 技术架构

- 💻 前端
  - Flutter
  - GetX 状态管理
  - Material Design 3

- 🧠 AI 能力
  - 豆包大模型
  - 知识图谱
  - 智能对话

- 📱 多平台支持
  - iOS
  - Android
  - Web
  - Desktop (Windows/macOS/Linux)

## 开发指南

### 环境配置

1. 安装依赖:
```bash
flutter pub get
```

2. 运行应用:
```bash
flutter run
```

### 项目结构

```
lib/
├── app/
│   ├── core/          # 核心功能
│   ├── data/          # 数据层
│   ├── modules/       # 功能模块
│   ├── presentation/  # 界面层
│   ├── routes/        # 路由配置
│   └── services/      # 服务层
├── generated/         # 生成的代码
└── main.dart         # 入口文件
```
